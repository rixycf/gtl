gtl
=====

make todo list

### Description
**gtl** is command line tool. It can make a todo listand write a process below TODO. **gtl** save TODO List in JSON file. It is put in the current directory.

### Demo


### Usage
add task  
`gtl add <task>`   

add process  
`gtl add -i < ID > -p <process> -p <process> -p ...` 

remove task  
`gtl remove <ID>`

done task  
`gtl done <ID>`

show list  
`gtl list`

shoe the detail  
`gtl list -d`
